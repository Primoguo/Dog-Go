# Dog Go PRD 文档集

本目录收录 Dog Go v3.0 原型重构及 M0.6 自主生活闭环相关的产品、视觉与实施规范。

## 建议阅读顺序

1. [`Dog_Go_PRD_v3_Prototype_Redesign.md`](Dog_Go_PRD_v3_Prototype_Redesign.md)  
   v3.0 主 PRD，定义产品愿景、信息架构、核心体验、功能范围与验收标准。
2. [`Dog_Go_M0.6_Autonomy_Proposal.md`](Dog_Go_M0.6_Autonomy_Proposal.md)  
   M0.6 自主生活闭环方案，定义需求、感知、意图、空间锚点和行动序列。
3. [`Dog_Go_Five_Scenes_Design_Guideline.md`](Dog_Go_Five_Scenes_Design_Guideline.md)  
   五个核心生活场景的定位、锚点、互动对象、行为和持久化痕迹规范。
4. [`Dog_Go_Visual_Design_Guideline.md`](Dog_Go_Visual_Design_Guideline.md)  
   栗子角色、场景、动画、UI 与资产生产的统一视觉规范。
5. [`Dog_Go_Scene_Production_and_Cross_Scene_Implementation.md`](Dog_Go_Scene_Production_and_Cross_Scene_Implementation.md)  
   从视觉概念稿进入分层资产、可运行场景及跨场景行为闭环的实施说明。

## 文档关系

- 主产品定义：v3.0 PRD
- 当前优先里程碑：M0.6「自主生活闭环」
- 产品与空间规范：五个核心生活场景设计理念
- 视觉基线：视觉设计理念与角色资产规范
- 生产落地：场景生产化与跨场景自主行为实施建议

现有 `docs/Dog Go v2.2 Native MVP PRD.md` 作为历史版本保留；本目录中的 v3.0 文档集代表后续原型重构方向。
