## 4. 真机演示脚本

1. 首次启动，阅读产品承诺并领养“栗子”。
2. 打开首次见面片段并选择一个回应。
3. 回到首页，进入 Debug 生活调试台并推进 20 分钟。
4. 打开“你不在时”，查看第一次短暂分别。
5. 再推进 8 小时或 24 小时，查看新增生活片段。
6. 打开“我们的日子”，确认里程碑、回应和“后来被想起”计数。
7. 开启 VoiceOver、较大动态字体和减少动态效果，复核核心流程。
8. 杀掉应用后重新打开，确认领养、事件、回应与记忆仍存在。
